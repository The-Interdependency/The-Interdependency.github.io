a0 — Agent Zero

This page is a bounded public summary of what is visible in The-Interdependency/a0 and its linked deployment materials. It describes repository structure and stated deployment intent without claiming that every described feature is complete, running, or validated.

What the public repository shows

  • Frontend: React, Vite, TypeScript, Tailwind, shadcn/ui.
  • Node layer: Express server handling auth, sessions, and API proxying.
  • Python layer: FastAPI backend with route modules, services, and storage layers.
  • Data layer: PostgreSQL-oriented configuration with Drizzle and SQLAlchemy-related files.
  • Deployment materials: Dockerfile, Cloud Build config, GitHub workflow directory, and deployment guide.

Public-facing interpretation

  • The repository reads as an active application codebase rather than a static document set.
  • Its architecture is more complex than the website and appears aimed at an interactive app environment.
  • The public docs include pricing, tiers, and deployment plans, but those should be read as implementation material unless separately confirmed in a live app.

What is useful for the website

  • A short architecture summary.
  • A link to the repository.
  • A clear distinction between repository claims, live app behavior, and canon language.
  • A consolidation page describing overlap with aimmh.
